    什么是皮尔逊相关系数？

    在统计学中，皮尔逊相关系数是用于度量变量直接的相关关系（线性相关），定义为两个变量之间的协方差和标准差的商，其值介于-1到1之间

    ♪ Tetrahedral angle

    什么是四面体角？

    水分子会和周围的两个水分子通过氢键形成一个如图的四面体网络，其中灰色的两点为周围水分子的氢原子。

    在水分子的四面体网络中，四面体角(TA)代表了水分子的排布规律。

    如果TA均匀分布在0-180°，即表征了液态水的无序状态。如果TA集中分布在90°，即表征此时水分子形成了高度有序的方形冰的结构。
