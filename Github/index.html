    ♪ 如何上传一个文件

    1. 创建一个仓库并复制它的地址。是否在此时创建Readme.md文件取决于你

    2. cd进入到上传内容的位置。当然Git bash here痛击这里是最好的选择

    3. 通过git init将这个项目变成一个Git可以管理的仓库

    4. 通过git add .将项目下的文件添加到仓库

    5. 通过git commit -m ""将项目提交到仓库，如果你想从历史记录里方便地找到改动记录的话，说明越有意义越好

    6. 通过git remote add origin 自己的仓库地址将本地仓库关联到Github上

    7. git push -u origin master将项目上传到Github上
